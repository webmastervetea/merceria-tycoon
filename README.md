# Juego de Simulador de una tienda de Merceria
## El proyecto consiste en crear un juego 2d web con metodologia de programacion Entity Component System (ECS) 
## Los graficos y animaciones utilizaran la metodologia Animación Procedural y por Código. El estilo visual sera Pintura Digital/HD (Mezcla de ambas).
## El juego debe de ser 100% compatible con moviles, tablets y televisiones Smart tv, y cualquier otro dispositivo
## Componentes del juego: El juego deber ser jugable en una sola pantalla.
### Tendra las siguientes plantas
* Planta 1 Mostrador: Aqui se atiende a los clientes, se puede vender productos del taller o del Almacen
* Planta 2 Taller: Aqui se fabrica los productos que se vende
* Planta 3 Almacen: Control de existencias con proveedores y compras de material
* Planta 4 Oficina: Aqui se controla todo lo relacionado con la parte administrativa, 
* Planta 5 Personal: Gestion de personal, posibilidad de contratar el personal
* Planta 6 I+D: 
* Planta 7 Banco: Ofrecera servicios que necesite la merceria como cuenta bancaria, pago, prestamos, tarjetas tpv ect..
* Planta 8 Logros: Iran apareciendo los diferentes logros que consigue el usuario
* Planta 9 Configuraciones o Ajustes: opciones de configuracion, como sonido on u off, sistema de guardado de la partida, modo oscuro modo claro y demas.

El juego debe de tener en cuenta los parametros de fustracion, Engagement Loop la jugabilidad y rendimiento
